# Portfolio Website

## Overview
This is my personal **portfolio website**, built to showcase my projects and professional experience. It is designed as an **online resume** where visitors can explore my work, view project details, and access links to source code and deployed applications. The website is built with **Next.js**, **Tailwind CSS**, **TypeScript**, and **Shadcn UI components**.

Visit my Portfolio here: https://my-portfolio-unp7.vercel.app

---

## Tech Stack
- **Framework**: Next.js
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **UI Components**: Shadcn

---

## Features
1. **Theme Toggler**
   - Supports light and dark modes for user preference.

2. **Project Gallery**
   - Displays a gallery of my projects with comprehensive details.
   - Includes screenshots, descriptions, and technologies used.
   - Provides links to GitHub repositories and live deployed sites.

3. **Online Resume**
   - Acts as a professional showcase of skills, experience, and projects.

4. **Responsive & Interactive UI**
   - Fully responsive design for desktop and mobile devices.
   - Interactive components enhance user experience.

---

## Architecture & Design
- **Next.js** provides server-side rendering and optimized routing.
- **Tailwind CSS** enables responsive and modular styling.
- **Shadcn Components** enhance UI consistency and aesthetics.
- **TypeScript** ensures type safety and maintainable code.
- **Theme Toggler** allows users to switch between light and dark modes.
- **Project Gallery** showcases projects with all relevant information, including screenshots and links.

---

## Summary
This portfolio website serves as an interactive online resume, highlighting my projects, skills, and professional work. The combination of Next.js, Tailwind, TypeScript, and Shadcn ensures a modern, responsive, and visually appealing experience for visitors.
